Bannertracking

Bannertracking - UXD/PRO1



Dit was een wat kleinere opdracht maar we moesten 1 pagina maken met een zelfgemaakte banner, die heb ik gemaakt via Adobe Photoshop en vervolgens in de webpagina gezet. Maar we moesten er voor zorgen dat als je op de banner zou klikken dat dit bijgehouden zou worden (het aantal clicks), dus hadden we ook backend code nodig. De backend code heb ik vooral van het internet gehaald omdat we toen nog weinig backend kennis hadden, dit heeft veel geholpen maar ik ben achteraf ook op een leraar afgestapt om vragen over de code te stellen zodat ik het beter kon begrijpen. Hiervan heb ik geleerd dat je op internet veel dingen kan overtypen maar het begrijpen van de code is natuurlijk ook heel belangrijk.
